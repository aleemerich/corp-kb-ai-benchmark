# Penetration testing **Produto:** AIRich Security Shield **Departamento:** Produtos **Versão:** 1.9 ## Requisitos Para utilizar o penetration testing corretamente, é necessário: 1. Ter uma conta ativa na plataforma AIRich 2. Possuir permissões adequadas no sistema 3. Estar com o plano de assinatura ativo 4. Utilizar um navegador compatível (Chrome, Firefox, Edge) ## Sobre Penetration testing O penetration testing do AIRich Security Shield representa uma parte fundamental da estratura de produtos da AIRich Tecnologia. Acesse Configurações > Penetration testing e siga as instruções apresentadas na interface. ## Contexto A AIRich Tecnologia mantém um compromisso contínuo com a evolução de seus produtos.
