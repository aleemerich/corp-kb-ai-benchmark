# Load testing > Documento técnico do AIRich DevOps Suite ## Requisitos Para utilizar o load testing corretamente, é necessário: 1. Ter uma conta ativa na plataforma AIRich 2. Possuir permissões adequadas no sistema 3. Estar com o plano de assinatura ativo 4. Utilizar um navegador compatível (Chrome, Firefox, Edge) ## Sobre Load testing O load testing do AIRich DevOps Suite representa uma parte fundamental da estratura de produtos da AIRich Tecnologia.
