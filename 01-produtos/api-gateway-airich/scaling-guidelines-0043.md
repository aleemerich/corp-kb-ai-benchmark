# Scaling guidelines > Documento técnico do AIRich API Gateway ## Escopo O escopo deste documento abrange todas as funcionalidades relacionadas a scaling guidelines no AIRich API Gateway, incluindo configurações, integrações e procedimentos operacionais. ## Requisitos Para utilizar o scaling guidelines corretamente, é necessário: 1. Ter uma conta ativa na plataforma AIRich 2. Possuir permissões adequadas no sistema 3. Estar com o plano de assinatura ativo 4. Utilizar um navegador compatível (Chrome, Firefox, Edge) ## Configuração A configuração do scaling guidelines pode ser realizada através do painel administrativo. Acesse Configurações > Scaling guidelines e siga as instruções apresentadas na interface. ## Contexto A AIRich Tecnologia mantém um compromisso contínuo com a evolução de seus produtos. O scaling guidelines passou por diversas melhorias nos últimos trimestres, incorporando feedback de clientes e avanços tecnológicos. ## Sobre Scaling guidelines O scaling guidelines do AIRich API Gateway representa uma parte fundamental da estratura de produtos da AIRich Tecnologia. Desenvolvido com foco em escalabilidade e performance, este módulo atende mais de 500 empresas clientes em todo o Brasil e internacionalmente. ## Objetivo Este documento tem como objetivo apresentar os aspectos técnicos e funcionais do scaling guidelines, fornecendo uma visão abrangente para as equipes de desenvolvimento, suporte e operações.
