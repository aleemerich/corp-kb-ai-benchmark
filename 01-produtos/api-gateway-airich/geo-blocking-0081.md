O geo-blocking passou por diversas melhorias nos últimos trimestres, incorporando feedback de clientes e avanços tecnológicos. ## Sobre Geo-blocking O geo-blocking do AIRich API Gateway representa uma parte fundamental da estratura de produtos da AIRich Tecnologia.
